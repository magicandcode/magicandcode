An Acceptable Appearance

An Acceptable Appearance

Presentation for PyLadies September 30 2020. This is the first of three parts in a series on how to achieve an acceptable Windows experience coming from MacOS (or Linux).

Download checklist as PDF

App & Text Size

  • Reduce app & text size

    Settings > System > Display or Settings > Ease of Access > Display

Desktop

  • Change desktop background

    Settings > Personalisation > Background or Settings > Personalisation > Themes > Background
    Ubuntu 16.10 background

  • Bin the Bin

    Settings > Personalisation > Themes > Related settings > Desktop icons settings

Colours

  • Set mouse pointer colour to black

    Settings > Personalisation > Themes > Background or Settings > Personalisation > Themes > Mouse Cursor Settings

  • Set dark mode and accent colour

    Settings > Personalisation > Colours or Settings > Personalisation > Themes

Lock Screen

  • Uncheck all and set background picture

    Settings > Personalisation > Lock Screen

Taskbar

  • Use small buttons

    Settings > Personalisation > Taskbar > Use small taskbar buttons [On]

  • Move to the top

    Settings > Personalisation > Taskbar > Taskbar location on screen [Top]

  • Remove redundant icons & buttons

    Settings > Personalisation > Taskbar > Combine taskbar buttons [Always hide labels]
    Settings > Personalisation > Taskbar > Select which icons to appear on the taskbar
    Right click on taskbar > Unselect “Show Cortana button” and “Show Task View button”

  • Add Show Desktop button

    Right click on desktop > New > Shortcut
    Set shortcut location: C:\Windows\explorer.exe shell:::{3080F90D-D7AD-11D9-BD98-0000947B0257}
    Set shortcut name: Show Desktop
    Right click on shortcut > Properties > Change icon…
    Send shortcut to Taskbar

Screen Saver

  • Install Fliqlo & set as screen saver

    Download Fliqlo for Windows
    Extract downloaded file and double click on Fliqlo Setup to install.
    Settings > Personalisation > Lock screen > Screen Saver Settings
