An Acceptable Appearance

Or how to make Windows look less like Windows

Presentation for PyLadies September 30 2020. This is the first of three parts in a series on how to achieve an acceptable Windows experience coming from MacOS (or Linux).

We'll start by fixing the appearance. It may be superficial but I can't bear the default look of Windows 10. Especially not after being used to work with a MacBook Pro. If you are of a similar disposition, fear not, there is hope! After changing a series of default settings you too can achieve an acceptable Windows appearance.

Note that this is a very opinionated piece but although your preferences don't align with mine, you can get some tips on how to achieve your acceptable Windows experience. And even if you do like the look of Windows there may be something here for you to improve the look further to your liking.

App & Text Size Desktop

App & Text Size

By default, the size of apps and text is set to (a lot) more than 100%, which doesn’t leave much room for applications, especially not vertically. Programmers are generally more concerned with screen height than width, since we want to see as many lines of code as possible. The overly large default settings in combination with the fact that most PC:s have a 16:9 display ratio (meaning they are more fitted to show HD video than code) is simply unworkable for a programmer. Especially if you’re used to the wonderful MacBook 16:10 Retina display. While we won’t achieve Retina status, we can change the default size.

Open Settings and go to Ease of Access (or System) and select Display, which should be the default selection. The setting is called Make everything bigger but we want to do the opposite and decrease the default percentage. The default value will depend on your device and the display size but generally you’ll always want to decrease from the default. Note that 100% may be too tiny depending on your display. I find 125% to be proficient and it makes a world of difference to my 14” device’s default of 150%.

Steps Checklist

  • Reduce app & text size

    Settings > Ease of Access > Display or Settings > System > Display

Desktop

Now that we hopefully can see better what we're doing it's time to fix the Desktop. You wouldn't think so going by my former MacBook desktop (...) but I dislike desktop icons. On Windows even more so.
But before we get into icons, you’ll want to change the typical default background image if you haven’t already.

The image varies depending on your PC’s manufacturer and if you like what you’ve got, by all means keep it. I've however gone all in for a background that does not remind me of Windows. The best strategy for this I find is to simply use the default background for another OS.
Currently, my personal go-to background is the Ubuntu 16.10 default background. It’s a perfect blend of subtlety and colour, and more importantly, it's decidedly not Windows-esque. I like this background so much that I also use with Ubuntu 18.

Once you've got a nice shiny background you don't want to clutter it with ugly icons, right? While one can get a custom icons theme, I find it's sufficient to just get rid of all of them. But unless you know how, it can be a very frustrating experience.
Personally I've tried to bin the Recycle Bin icon more times than I'm willing to admit. Someone I know resigned and dragged it to a remote corner of the screen.
However the good news is that you can get rid of it for good; Bin the Bin once and for all.

While in Settings go to Personalisation > Themes. Scrolling down there should be a header Related settings and a link to Desktop icons settings. This opens a new dialog where you're presented with available icons to display on the Desktop. Each icon has a checkbox and by unchecking all you'll finally have that clutter free Desktop I've always dreamed of.
This is also where you cange icon images if you're so inclined. Make sure that Allow themes to change desktop icons is selected. Then you can change the icons by selecting a new theme. I've never used a theme (other than the custom theme implicitly created by making changes) so I won't discuss it further. But it's an area to explore if the resulting appearance after going through all the steps isn't satisfactory.

Steps Checklist

  • Change desktop background

    Settings > Personalisation > Background or Settings > Personalisation > Themes > Background
    Ubuntu 16.10 background

  • Bin the Bin

    Settings > Personalisation > Themes > Related settings > Desktop icons settings
